API REST with Spring Boot MVC + Angular 8/9 + JWT Authentication


This system was developed in Angular and accesses an API developed in Java with Spring Boot, MVC design pattern and Spring Security (Jason Web Token Authentication). Requests are sent to the API to save and retrieve data written to the database. Before logging into the system, the user is validated using a token and passwords are recorded using encryption.

It is possible to create records and associated telephone numbers, search for records saved in the system and generate a comparative graph. Each record has a password that allows you to log into the system. A service was created to recover the password when logging into the system. This password is sent by email, however it is necessary to assign a valid email address when creating a new record. The hosting server has limitations and unfortunately the WAR FILE exceeded the maximum accepted limit by the server and therefore a feature was removed from the system.

This way it is not possible to generate PDF, the jasper reports libraries caused a significant increase in the size of the war file. Configurations to increase this limit have already been tested, and the solution to this problem is being analyzed daily so that this feature works again soon, as in tests carried out locally, where this limitation does not occur. As well as improvements in the code are constantly being analyzed so that problems do not occur when using the application.

Other technical information: Spring Tool Suite, VS Code, Postgres 15.4, Bootstrap 5.0, EC2 Amanzon Tecnology, Putty, File Zilla, DBeaver.
